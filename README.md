# CodeMirror + Matter.js プレイグラウンド

ブラウザ上でJavaScriptコードを書いて、2D物理シミュレーションをリアルタイムに動かせる学習用ツールです。

## 概要

画面左側のコードエディタにコードを入力すると、右側のキャンバスに物理オブジェクト（円・長方形・ドミノなど）が描画されます。コードを1行ずつ順番に実行するステップ実行モードにも対応しており、プログラムの動作を視覚的に確認できます。

## ファイル構成

```
puppy_project/
├── index.html   # UI・レイアウト・ライブラリ読み込み
├── main.js      # エディタ・物理エンジン・実行ロジック
└── README.md    # このファイル
```

## 使い方

1. `index.html` をブラウザで開く
2. 左側のエディタにコードを入力する
3. **🔁 実行** ボタン または `Ctrl + Enter` でコードを実行する
4. **❌ 全削除** ボタンでキャンバスをリセットする
5. ドロップダウンからサンプルプログラムを選んで試すこともできる

## 使える関数

| 関数 | 説明 |
|------|------|
| `Circle(x, y, options)` | 指定座標に円を生成する |
| `Rectangle(x, y, options)` | 指定座標に長方形を生成する |
| `createDominos(count, startX, startY, spacing)` | ドミノを一列に並べる |
| `startDomino()` | 最初のドミノに力を加えて倒し始める |

### Circle / Rectangle のオプション

```js
Circle(300, 100, { radius: 20, color: "red", restitution: 0.5 });
Rectangle(300, 200, { width: 60, height: 40, color: "blue" });
```

## サンプルプログラム

- **ドミノ倒し** — ドミノを並べて連鎖的に倒す
- **色付き円** — 異なる色・サイズの円を一括生成する
- **色付き円（1行ずつ）** — 円を1行ずつステップ実行で生成する

## 実行モード

| モード | 説明 |
|--------|------|
| `safeEval` | コード全体を一括で実行する |
| `step` | 1行ずつ800ms間隔で実行し、現在の行をハイライト表示する |

## 使用ライブラリ

- [CodeMirror 5](https://codemirror.net/5/) — コードエディタ
- [Matter.js 0.19.0](https://brm.io/matter-js/) — 2D物理エンジン
